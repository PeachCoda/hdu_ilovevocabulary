# hdu_ilovevocabulary
杭电我爱记单词

包含完整的可视化控制面板，支持本地题库 (JSON) 导入、AI 兜底配置、中英双向查词与原生强力模拟点击。

## 🚀 安装指南

1.  **准备环境**：确保浏览器已安装 **[Tampermonkey (油猴)](https://www.tampermonkey.net/)** 扩展插件。
2.  **一键安装**：点击下方链接，打开脚本页面，点击绿色的“安装”按钮：
    > **[👉 点击此处安装 (Greasy Fork 商店) 👈](https://greasyfork.org/zh-CN/scripts/571267-hdu%E6%88%91%E7%88%B1%E8%AE%B0%E5%8D%95%E8%AF%8D)**

## 📖 使用教程

### 第一步：环境伪装

如果你的考试系统提示 **“请在手机端考试”**，请务必执行此步：

1.  在答题页面按下 **`F12`** 打开开发者工具。
2.  按下快捷键 **`Ctrl + Shift + M`**（或点击左上角手机和平板重叠图标）进入手机模拟模式。
3.  在顶部工具栏机型下拉菜单中选择一个具体的型号（如 **`iPhone 12 Pro`**）。
4.  **【最重要】按下 `F5` 刷新页面**。拦截提示将消失，脚本会自动加载。

### 第二步：导入题库

1.  在右上角的脚本控制面板中，点击 **“📂 导入题库 (JSON)”**。
2.  选择你本地包含大量词汇匹配的 JSON 文件。
3.  看到面板上“题库数量”更新即说明导入成功。

### 第三步：配置AI

1.  在面板的“🤖 AI 配置”区域填写：
    * **API 地址**：建议使用 DeepSeek (`https://api.deepseek.com/v1`) 或其他兼容 OpenAI 接口的中转服务。
    * **API 密钥**：填写你的 API Key (sk-...)。
2.  勾选 **“启用 AI 兜底”** 并点击 **“💾 保存配置”**。
3.  看到“AI 状态”变为绿色的“✓ 已启用”即可。

---

## ⚠️ 警告
本脚本没有设置本地题库不匹配时的跳过机制，如果没有配置AI兜底则脚本大概率会失效。

**在不能保证题库覆盖率的情况下请务必配置AI！**
